#考试系统
备考系统

完全本地化持久存储 (localStorage)：在网页中做出的所有勾选、新增、修改、删除等操作都会自动保存在当前浏览器中，刷新或关闭页面后数据不会丢失。

可视化智能时间轴（左侧）：自动抓取所有课程的考试时间，按先后顺序进行自适应倒序重排。内置状态机自动根据考试时间的紧迫程度切换高亮（距离考试 <24h 为红色紧急状态，<3天为橙色活跃状态，未开始为蓝色安全状态，已结束为灰色并降低不透明度）。

备考双板块精细深挖：

知识点树状盘点：支持自由创建“章/大板块”，并在各章下无限细化具体“细分知识点”；

刷题高频考法跟踪：支持根据具体学科定制“期末试卷题型”（如单选、计算、论述大题），并在该题型下配置“具体考法/命题切入方向”。

全自动化进度看板（顶部）：包含总报考科目数、当前最近一场要考的焦点科目及精准倒计时、全科总备考进度条以及秒级同步的系统时钟。

额外功能扩展：跨设备备份与恢复 (JSON)：右上方内置了“导出备份”和“导入数据”按钮。您可以将所有复习安排一键打包成 .json 文件导出，并在其他设备或浏览器上完美还原。
